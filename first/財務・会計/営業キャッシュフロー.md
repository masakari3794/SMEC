##　キャッシュフロー

### 営業キャッシュフロー

- 基本的な考え方
  - 税引前当期純利益＋減価償却費±売上債権の増減±棚卸資産の増減±仕入債務増減-法人税等の支払額
- 具体例
  - 損益計算書（P/L）
  - | 項目                     | 額（単位：千円） |
    | ------------------------ | ---------------- |
    | 売上高                   | 1,000,000        |
    | 営業利益                 | 200,000          |
    | 法人税、住民税及び事業税 | 60,000           |
    | 当期純利益               | 140,000          |
  - 貸借対照表（B/S）
  - | 項目         | 前期末（単位：千円） | 当期末（単位：千円） |
    | ------------ | -------------------- | -------------------- |
    | 売掛金       | 50,000               | 46,000               |
    | 棚卸資産     | 30,000               | 33,000               |
    | 買掛金       | 35,000               | 36,200               |
    | 未払法人税等 | 30,000               | 30,000               |
  - 計算
    1. 税引前当期純利益を見る
    - これは営業利益と同じこと
    - 表から200,000
    2. 減価償却費を足す　
    - 減価償却費は費用だが現金は出ていないので足し戻す
    - 200,000+30,000=230,000
    3. 売掛金の増減
    - 前期末50,000、当期末46,000→4,000減少
    - 売掛金が減る＝売上代金を回収した→現金が増える→+4,000
    - 230,000＋4,000=234,000
    4. 棚卸資産の増減
    - 前期末30,000、当期末33,000→3,000増加
    - 棚卸資産が増える＝商品を仕入れて在庫が増えた→現金が出ていっている→-3,000
    - 234,000-3,000=231,000
    5. 買掛金の増減
    - 前期末35,000、当期末36,200→1,200増加
    - 買掛金が増える＝まだ払っていない仕入が増えた→現金支出を抑えている→+1,200
    - 231,000＋1,200=232,200
    6. 法人税等の支払額
    - 損益計算書では「法人税、住民税及び事業税」が60,000
    - 貸借対照表では「未払法人税等」に関して、前期末30,000、当期末30,000
    - 未払いが増減していない→実際に払った法人税等=当期計上額60,000→-60,000
    - 232,200-60,000=172,200

### 補足
- 有形固定資産の売却による収入は、投資活動のキャッシュフローとして扱われる
