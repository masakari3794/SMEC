## 投資意思決定

### 設備投資案の優劣を評価する方法

| 方法                     | 例                                                                   |
| ------------------------ | -------------------------------------------------------------------- |
| 時間価値を考慮する方法   | 正味現在価値（NPV）<br />内部収益率法（IRR）<br />収益性指数法（PI） |
| 時間価値を考慮しない方法 | 回収期間法（PP）                                                     |

- 回収期間法は回収期間後のキャッシュフローを無視している

### 正味現在価値（NPV）

- 今の価値で見て、どれだけ儲かるか
- この現在価値から初期投資を引くことで正味投資価値が計算される
- この値が0より大きければ採択（投資のGoサイン）と考える

### 割引現在価値
  - `割引現在価値（PV） = 将来キャッシュフロー ÷ (1 + 割引率)^年数`
  - （永続的なキャッシュフローがある場合）`キャッシュフロー/割引率（資本コスト）`


### 確実性等価法

- 本来的には不確実性のある期待キャッシュフローをその不確実性の度合いによって、確実性の高いキャッシュフローに変換し正味現在価値を求める手法
- 確実性等価法で用いられる割引率はリスクのない収益率（例えばリスクフリーレート）

### リスク調整割引率法

- 期待キャッシュフローの不確実性を割引率で考慮して正味現在価値を求める手法
- つまり、分母の割引率に不確実性に応じたリスクプレミアムを加算して割引率を大きくする


### 内部収益率法

- 内部収益率（IRR）
  - NPVが0になる割引率（資本コスト）
  - これが実際の割引率（資本コスト）より大きければ採択

### 内部収益率に関して

- 内部収益率が最低限必要とされる収益率である資本コストよりも大きければ、その投資案は有利であるから採用すべきと判定する
- 投資案の内部収益率と資本コストを比較する際に、資本コストと全社的加重平均資本コスト（WACC）の二つのコストが与えられた場合
  - WACCは全社の平均的なリスク特性を反映したものとして推定されている
  - よって、検討対象である投資案が全社の平均的事業と類似したリスク特性を持っている場合にはWACCをそのまま適用する
  - しかし、事業のリスク特性が異なる分野の投資案に対しては適用しない
  - 事業のリスク特性が異なることの確認は、資本コストの違いを見ればいい
  - 回収期間が長くても、内部収益率が高くなる場合もある
  - 内部収益率法の問題点として、投資によって生じる毎年のキャッシュフローの符号が複数回変化する場合、異なるいくつかの値が得られる場合がある
