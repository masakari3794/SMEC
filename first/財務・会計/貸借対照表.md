## 貸借対照表

- 会社が今どれだけお金を持っていて、それをどう集めて、どう使っているかを表す表
- 前提条件
  - 資産（左側）＝負債＋純資産（右側）
- 貸方と借方との関係
  - 資産が増えるor負債・純資産が減る（借方）、負債・純資産が増えるor資産が減る（貸方）
  - | 借方に記載する取引の内容     | 貸方に記載する取引の内容     |
    | ---------------------------- | ---------------------------- |
    | 資産の増加                   | 資産の減少                   |
    | 負債の減少                   | 負債の増加                   |
    | 純資産の減少                 | 純資産の増加                 |
    | 費用の増加                   | 費用の減少                   |
    | 収益の減少（税金が増える等） | 収益の増加（税金が減った等） |
- 資産
  - お金をどう使っているか
  - 主な中身
    - 現金・預金、売掛金（まだ回収していないお金）、商品・在庫、建物・設備
    - 会社が持っている価値あるもの
  - 資産は、流動資産（1年以内に現金化できるもの）、固定資産（1年以内に現金化できないもの）に分けられる
    - 流動資産
      - 売掛金は、通常の営業取引によって生じた債権であり正常業務循環基準が適用され、代金回収期間の長短に関わらず流動資産に分類される
      - 棚卸資産にも正常業務循環基準が適用され、販売までの期間
    - 固定資産
      - 減損会計の適応対象資産となる
      - 無形固定資産
        - 具体的な形態を持たない資産のこと
          - ポイント
            - 人的資産については、企業が保有している価値ではあるが、恣意性の介入により資産としての客観的な評価ができないことなどから、無形固定資産に計上されない
            - 法律上の権利である特許権、商標権などは、法定償却期間にわたり、残存価値をゼロとする**定額法**により償却し、のれんについては、20年以内のその効果の及ぶ機関にわたって、**定額法その他の合理的な方法により規則的に償却する**
- 負債
  - 返さないといけないお金
  - 例
    - 借入金・買掛金・未払費用
- 純資産
  - 返さなくていいお金
  - 例
    - 資本金、利益剰余金
- 細かいルール
  - 貸倒引当金が複数のところ（売掛金や短期貸付金（1年以内に返す必要がある貸付金））でそれぞれあったとしてもまとめて表記できる
    - 貸倒引当金は資産のマイナス扱い（資産の欄に記載する）
  - 繰延税金資産に関して（借方）
    - 将来の税金が減る権利
    - 売るわけでもなく、現金になるわけでもないので固定資産扱い
    - 貸方には法人税等調整額となる
  - 資産除去債務に関して
    - 将来、撤去・原状回復しなきゃいけない費用
      - 例）工場を建てたので、将来的に取り壊して土地を元に戻す必要がある　その費用を今の時点で見積もっておく
    - 資産側に建物が、負債側に資産除去債務がくる
  - 不動産の扱いに関して
    - オフィスや工場のように自社で使う不動産は、有形固定資産になる
    - 一方で、不動産業者が取り扱う不動産は売るものなので、棚卸資産（在庫）扱いとなる
