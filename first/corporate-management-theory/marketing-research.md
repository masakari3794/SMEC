## マーケティング・リサーチ

* リサーチの種類

  * 探索的リサーチ

    * アイデアと洞察を得ることが目的
  * 検証的リサーチ

    * ある特定の仮説を検証し、因果関係を調べることを目的としている
  * インサイトリサーチ

    * 回答者の潜在意識下にあるようないわゆる本音を探り出す調査方法
* コーディング

  * 発言内容や記述内容に関して代表的な言葉や意見内容をまとめてカテゴリーに分類しコード（符号）変換する
* マーケティングにおけるデータ

  * 一次データ

    * ある木鉄器のために実験や調査を行うことによって新規に収集されたデータ
  * 二次データ

    * 他の目的のために既に収集されているデータであり、組織内に蓄積されている内部データと図書館や業界団体といった組織外に存在している外部データがある
* ジョハリの窓

  * |                    | 自分でわかっている | 自分ではわからない |
    | ------------------ | ------------------ | ------------------ |
    | 他人が分かっている | 開放の窓           | 盲点の窓           |
    | 他人はわからない   | 隠された窓         | 未知の窓           |
  * マーケティング・リサーチとの関係
    * インタビュー調査
      * どの窓の自己データの収集についても有効といえる
    * 行動観察調査
      * 他者が観察者として個人の行動を記録しその行動に基づいてフィードバックを提供する手法であり、調査対象者の行動や反応を直接調査者が観察することができるので、調査者も調査対象者もよく知っている領域を拡大することに貢献する
      * 「開放の窓」「盲点の窓」についてのデータを得るために有効な調査手法
      * 観察だから他者が分かっているわけだ
      * 主観的な自己認識や相手からのフィードバックが必要
    * 定量的なアンケート調査
      * 客観的なデータを得るために使用され鵜r
