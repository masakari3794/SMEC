## 制度的同型化

* [参考文献](https://handbook.bizsci.net/isomorphism/)

|              |              | メカニズム=同型的組織変化の源泉                                                                                                                                                    |
| ------------ | ------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 競争的同型化 |              | 環境の淘汰圧力のようなメカニズム。（簡単に言ってしまえば、生存競争をして適者生存で生き残ったものは似てくるということ）                                                             |
|              | 強制的同型化 | 依存している組織からの圧力、社会の中での文化的期待、例えば、法的な規制。                                                                                                           |
| 制度的同型化 | 模倣的同型化 | 組織はより正統的あるいは、より成功していると認識している類似の組織を後追いしてモデル化する。不確実性は模倣を助長する。                                                             |
|              | 規範的同型化 | 主に職業的専門化に起因するもので、①大学の専門家による公式の教育と正統化、②職業的ネットワークの成長と洗練が重要。<br />横断的な組織のネットワーク。人員の選別も重要なメカニズム。 |
