## 機械的管理システム・有機的管理システム

| 機械的管理システム                             | 有機的管理システム                                                         |
| ---------------------------------------------- | -------------------------------------------------------------------------- |
| 機能的タスクの専門分化・分割                   | 共通のタスクに対し、異なる知識・経験を基礎とする専門化                     |
| 各タスクの抽象性（全体目標や技術と関係が遠い） | 各タスクの具体性（全体状況に結びついている）                               |
| 直属の上司による各成果の調整                   | 横の相互作用を通じた各タスクの調整・再定義                                 |
| 各役割の職務・権限及び方法の明確化             | 責任を限られた領域に限定しない（問題を他者の責任にしない）                 |
| 職務・権限・方法が機能的地位の責任に変換される | 技術的規定を超えたより広い関心へのコミットメント                           |
| 制御・権限・伝達の階層的構造                   | 制御・権限・伝達のネットワーク型構造                                       |
| 階層トップへの知識の集中による階層構造の強化   | ネットワーク内での知識の分散、権限・伝達の中心はアドホック（臨時的）に変化 |
| メンバー間の垂直的相互作用（上司ー部下）       | より水平的相互作用、異なる地位間の伝達は命令的ではなく指導的               |
| 上司への指示・命令に支配された職務             | 情報提供と助言内容のコミュニケーション                                     |
| 組織への忠誠と上司への服従の協調               | タスクそのものと優れた仕事をしようとする精神へのコミットメント             |
| 組織内特有の知識・経験・スキルの強調           | 組織外の専門家集団でも通用する専門能力およにそうした集団への参加の強調     |
